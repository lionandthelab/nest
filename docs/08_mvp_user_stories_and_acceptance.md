# MVP 사용자 스토리 및 수용 기준

## 1) 관리자 스토리

### US-ADM-01 홈스쿨 개설
- As a 관리자, I want 홈스쿨을 생성해서 운영을 시작하고 싶다.
- 수용 기준
- 홈스쿨 이름/타임존 입력 후 생성 가능
- 생성자는 자동으로 `HOMESCHOOL_ADMIN`
- 초대 코드가 자동 발급

### US-ADM-02 학기와 반 생성
- As a 관리자, I want 학기와 반을 생성해 학사 운영 단위를 준비하고 싶다.
- 수용 기준
- 학기 시작/종료일 필수
- 반 이름/정원 입력 가능
- 아카이브 학기는 수정 불가

### US-ADM-03 채팅 기반 시간표 생성
- As a 관리자, I want 프롬프트 입력으로 시간표 초안을 빠르게 받고 싶다.
- 수용 기준
- 프롬프트로 생성안 조회 가능
- 생성안별 충돌/경고/설명 확인 가능
- 전체/부분 적용 가능

### US-ADM-04 수동 드래그앤드롭 편집
- As a 관리자, I want 생성안을 직접 끌어다 놓아 세밀하게 수정하고 싶다.
- 수용 기준
- 카드 이동 즉시 충돌 검증
- 동일 반-동일 시간 중복 차단
- 동일 교사-동일 시간 중복 차단

### US-ADM-05 Google Drive 연동
- As a 관리자, I want 기관 Google Drive를 연결해서 미디어를 중앙 관리하고 싶다.
- 수용 기준
- OAuth 연결 성공 후 상태 확인 가능
- 루트 폴더 지정 가능
- 연결 해제 가능

## 2) 부모 스토리

### US-PAR-01 내 아이 반 시간표 조회
- As a 부모, I want 내 아이 반 시간표를 쉽게 보고 싶다.
- 수용 기준
- 주간 보기 제공
- 내 아이가 속한 반만 조회
- 최신 변경 내역 반영

### US-PAR-02 내 아이 갤러리 조회
- As a 부모, I want 내 아이 관련 사진/영상을 보고 싶다.
- 수용 기준
- 기간/수업/반 기준 필터 가능
- 권한 밖 아이 미디어 조회 차단

## 3) 교사 스토리

### US-TCH-01 수업 계획표 등록
- As a 교사, I want 담당 세션별 계획표를 작성하고 싶다.
- 수용 기준
- 목표/준비물/활동 필드 입력 가능
- 담당 세션에만 작성 권한

### US-TCH-02 활동 기록 작성
- As a 교사, I want 학생 활동을 기록하고 싶다.
- 수용 기준
- 출석/관찰/과제 유형 선택
- 담당 학생에만 기록 가능

### US-TCH-03 사진/영상 업로드
- As a 교사, I want 현장에서 바로 촬영해 업로드하고 싶다.
- 수용 기준
- 촬영 후 업로드 세션 생성 가능
- 업로드 완료 시 Drive 파일 ID 저장
- 실패 시 재시도 가능

## 4) 시스템 스토리

### US-SYS-01 감사 로그
- As a 시스템 운영자, I want 주요 변경 이력을 추적하고 싶다.
- 수용 기준
- 시간표/배정/권한/Drive 설정/미디어 변경 로그 생성
- 변경자/시점/변경 전후 데이터 저장

## 5) E2E 핵심 시나리오

- 시나리오 A
- 관리자 로그인 → 학기 생성 → 채팅 편성안 생성 → 부분 적용 → 드래그 수정 → 시간표 확정
- 기대 결과: 충돌 없는 시간표 확정 및 알림 이벤트 생성

- 시나리오 B
- 교사 로그인 → 수업 중 사진 촬영 → 업로드 완료
- 기대 결과: Drive 반영 + 갤러리 조회 가능

- 시나리오 C
- 부모 로그인 → 내 아이 시간표 조회 → 내 아이 갤러리 조회
- 기대 결과: 권한 범위 데이터만 노출

## 6) 우선순위

- P0: 인증, 권한, 학기/반/시간표, 채팅+수동 편성, 충돌검증
- P1: 교사 계획표/활동 기록, Drive 연동, 미디어 업로드/갤러리
- P2: Discovery/Nest Community/Ttobak 기본형
