# Nest 제품 비전 및 브랜드 가이드

## 1) 브랜드 정체성

- 앱 이름: **Nest (네스트)**
- 브랜드 문장: **"우리 아이가 날아오르기 전, 따뜻한 둥지 Nest"**
- 핵심 가치
- 따뜻한 환대
- 정서적 안정
- 아이 중심 성장
- 가정-교사-공동체의 동반 양육

## 2) 컬러 시스템 (Warm Nest)

### Main
- `Dusty Rose` `#DCAE96`
- 의미: 인간의 온기, 정서적 안정, 부드러운 지지
- 사용: 주요 CTA, 핵심 강조 영역, 상단 브랜드 바

### Base
- `Creamy White` `#F9F7F2`
- 의미: 아이들의 가능성이 자유롭게 펼쳐지는 따뜻한 배경
- 사용: 페이지 배경, 카드 배경, 학습/활동 화면 기본 캔버스

### Point
- `Deep Wood` (권장 예시 `#5E4636`)
- `Muted Sage` (권장 예시 `#7D9272`)
- 의미: 뿌리, 토양, 생명력, 지속 성장
- 사용: 보조 버튼, 상태 뱃지, 시각 구획, 그래프 포인트

## 4) 핵심 서비스 모듈

### Discovery (발견)
- 목적: 아이의 기질/강점/잠재력 진단
- MVP 기능
- 검사 템플릿 제공
- 결과 리포트(강점/관찰 포인트/권장 활동)
- 학부모와 담임 교사 공유

### Nest Community (안식처)
- 목적: 부모 커뮤니티 및 상호 지원
- MVP 기능
- 공지/질문/경험 공유 게시판
- 학기/반 기반 소모임 채널
- 관리자 공지 핀 고정

### Ttobak (또박)
- 목적: 아이 맞춤형 아날로그 학습지 생성
- MVP 기능
- 난이도/주제/학습목표 기반 템플릿 생성
- PDF 출력
- 학습지 생성 이력 및 재사용

### Nest Gallery
- 목적: 수업 현장 사진/영상의 안전한 공유
- MVP 기능
- 선생님 촬영 후 Google Drive 자동 업로드
- 학부모/선생님 권한 기반 갤러리 조회
- 반/아이/수업 기준 필터

## 5) 제품 포지셔닝

- Nest는 단순 LMS가 아니라 **운영(행정) + 관계(공동체) + 성장기록(미디어/활동)**을 통합한 홈스쿨링 운영 플랫폼이다.
- 1차 우선순위는 행정 안정성(학기/반/시간표/교사 배정)이다.
- 시간표는 **채팅 프롬프트 생성 + 수동 드래그앤드롭 보정**을 함께 제공해 운영 효율과 통제력을 동시에 확보한다.

## 6) 브랜드 톤 가이드

- 문장 톤: 따뜻하고 명확하며 비판단적
- 정보 전달: 간결하지만 실행 가능해야 함
- 마이크로카피 원칙
- "관리"보다 "돌봄" 맥락 우선
- 실패 안내 시 비난 금지, 대안 먼저 제시
- 부모/교사/관리자 모두를 협력 파트너로 호칭
