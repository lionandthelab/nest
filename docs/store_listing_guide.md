```
Nest는 홈스쿨링 가정을 위한 통합 운영 플랫폼입니다.

우리 아이가 날아오르기 전, 따뜻한 둥지에서 함께 성장하는 경험을 만들어갑니다.

[주요 기능]

1. 학기 운영 관리
- 학기/반/과목을 한눈에 설정하고 관리
- 가정/아이/교사 배정을 체계적으로 운영
- 교실 배정과 운영 현황을 실시간으로 파악

2. 시간표 스튜디오
- 드래그앤드롭으로 직관적인 시간표 편성
- 교사/교실 충돌을 자동으로 감지하고 방지
- 과목/교사/교실 팔레트에서 간편하게 배정
- 시간표 확정 및 변경 이력 관리

3. 학부모 전용 뷰
- 내 아이 기준의 맞춤형 시간표 조회
- 학습 현황 및 활동 기록 확인
- 홈스쿨 공지사항을 바로 확인

4. 교사 전용 허브
- 담당 반/수업 기준으로 운영 정보 집중
- 학기 계획표 작성 및 아동 활동 기록
- 수업 자료와 미디어 관리

5. 커뮤니티
- 홈스쿨 구성원 간 소통 공간
- 게시글/댓글/좋아요로 활발한 교류
- 관리자 모더레이션 기능으로 안전한 커뮤니티

6. 갤러리
- Google Drive 연동으로 수업 사진/영상 안전하게 보관
- 반/아이/수업 기준 필터로 쉽게 검색
- 권한 기반으로 안전한 미디어 접근

- 홈스쿨에 참여하는 학부모
- 홈스쿨에서 가르치는 교사

[특징]
- 역할 기반 맞춤 화면 (관리자/부모/교사)
```

## 4. 빌드 체크리스트

### Android (AAB)
```bash
cd frontend
# 1. key.properties 설정 확인
# 2. 빌드
flutter build appbundle --release
# 출력: build/app/outputs/bundle/release/app-release.aab
```

### iOS (Archive)
```bash
cd frontend
# 1. Xcode에서 Runner.xcworkspace 열기
# 2. Team / Signing 설정
# 3. Product > Archive
# 4. Distribute App > App Store Connect
```
