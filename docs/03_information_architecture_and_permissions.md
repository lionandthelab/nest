## 3) 권한 모델

### 역할 정의
- `HOMESCHOOL_ADMIN`: 운영 전권 + Drive 연동 관리
- `PARENT`: 본인 가정/자녀 중심 조회
- `TEACHER`: 담당 수업 및 담당 학생 정보 접근 + 미디어 업로드
- `GUEST_TEACHER`: 초청된 범위 교사 권한 + 미디어 업로드
- `STAFF`: 제한된 운영 지원 권한(옵션)

## 4) 접근 정책 상세

- 부모는 본인 가정에 소속된 아이 데이터/갤러리만 조회 가능
- 교사는 본인 배정 세션/반의 아이 데이터/갤러리만 조회 가능
- 초청 교사는 계약 기간 또는 학기 종료 시 접근 자동 만료
- 모든 민감 데이터 접근은 Audit 로그 기록

## 5) 멀티 테넌시 정책

- 모든 데이터는 `homeschool_id` 기준 논리 격리
- 사용자 계정은 복수 홈스쿨 소속 가능
- 토큰에는 활성 홈스쿨 컨텍스트 포함

## 6) 알림 정책 (MVP)

- 시간표 확정/변경 시 대상자 알림
- 교사 배정 변경 시 해당 교사 알림
- 미디어 업로드 완료 시 반/가정 대상 알림(옵션)
- 학기 시작 7일 전/1일 전 리마인드
