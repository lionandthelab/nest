## 3. 브랜드 톤 & 비주얼

### 3.1 슬로건
> **"우리 아이가 날아오르기 전, 따뜻한 둥지"**

### 3.2 컬러 팔레트

| 역할 | 컬러 | 코드 | 용도 |
|------|------|------|------|
| Primary | Dusty Rose | `#DCAE96` | CTA 버튼, 강조 |
| Background | Creamy White | `#F9F7F2` | 페이지 배경 |
| Text/Accent | Deep Wood | `#5E4636` | 헤드라인, 아이콘 |
| Nature Accent | Muted Sage | `#7D9272` | 뱃지, 성공 상태 |

### 3.3 톤 앤 매너
- 따뜻하고 친근한 어조 (아이를 키우는 부모의 마음)
- 기능 나열보다 **감정적 가치** 강조
- 전문적이되 딱딱하지 않은 표현
- 한국어 기본, 영문 병기 최소화

### Section 3: 주요 기능 쇼케이스 (Feature Showcase)

좌우 교차 레이아웃 (이미지 + 텍스트)

#### 3-1. 스마트 시간표 스튜디오
- **비주얼**: 시간표 편집 화면 스크린샷
- **카피**:
  - "반별 시간표를 한 화면에서 관리하세요"
  - 교사 충돌 실시간 감지
  - PNG 내보내기로 학부모 공유
  - 교실 상황표 자동 생성

#### 3-2. 학부모 허브
- **비주얼**: 부모 뷰 스크린샷 (아이 선택 + 시간표)
- **카피**:
  - "아이를 선택하면 이번 주 시간표가 바로"
  - 교사가 기록한 활동 일지 열람
  - 공지사항 실시간 확인

#### 3-3. 교사 워크스테이션
- **비주얼**: 교사 허브 스크린샷
- **카피**:
  - "수업 계획부터 활동 기록까지 한 곳에서"
  - 학기 계획표 작성
  - 학생별 관찰·과제·출석 기록

#### 3-4. 커뮤니티 & 갤러리
- **비주얼**: 커뮤니티 피드 + 갤러리 스크린샷
- **카피**:
  - "우리 홈스쿨만의 SNS"
  - 공지 고정, 신고·관리 기능
  - Drive 연동 사진 앨범
