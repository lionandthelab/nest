Last updated: 2026-03-13

이 문서는 Google Play Store 및 Apple App Store 심사 제출 시 필요한 질문지 답변, 등급 설정, 거절 사유 예방, 테스트 계정 준비를 위한 실무 가이드입니다.

Nest는 교육 및 홈스쿨 관리 앱입니다. 아래 답변을 기준으로 질문지를 작성합니다.

| 만화/판타지 폭력이 있나요? | 아니오 | 없음 |
| 현실적인 폭력이 있나요? | 아니오 | 없음 |

| 질문 | 답변 | 근거 |
|------|------|------|

| 누드 이미지가 있나요? | 아니오 | 없음 |

### 기타
| 질문 | 답변 | 근거 |
|------|------|------|
| 도박 요소가 있나요? | 아니오 | 없음 |

| 질문 | 답변 | 근거 |
|------|------|------|

- **Google Play IARC 예상 등급**: 전체 이용가 (Everyone)
- **Apple App Store 예상 등급**: 4+

## 2. Google Play 데이터 안전 섹션 (Data Safety)

Play Console > 앱 콘텐츠 > 데이터 안전에서 아래 내용을 입력합니다.

### 데이터 수집 여부
- **앱에서 사용자 데이터를 수집하거나 공유하나요?** → 예

### 수집 데이터 유형 및 목적

| 데이터 유형 | 수집 여부 | 공유 여부 | 사용 목적 |
|-------------|-----------|-----------|-----------|
| 이메일 주소 | 예 | 아니오 | 계정 관리, 앱 기능 |
| 이름 | 예 | 아니오 | 앱 기능 (프로필 표시) |
| 사용자 ID | 예 | 아니오 | 앱 기능 (인증) |
| 사진 및 동영상 | 예 (선택) | 아니오 | 앱 기능 (갤러리) |
| 게시글 및 댓글 (텍스트) | 예 (선택) | 아니오 | 앱 기능 (커뮤니티) |
| 앱 활동 (접속 로그) | 예 | 아니오 | 분석, 보안 |

### 보안 관행
| 항목 | 답변 |
|------|------|
| 데이터 전송 시 암호화 | 예 (TLS) |
| 사용자가 데이터 삭제를 요청할 수 있나요? | 예 (계정 삭제 기능 제공) |
| 아동 지향 서비스인가요 (COPPA/GDPR-K)? | 아니오 (성인 운영자 대상) |
| 독립적인 보안 감사를 받았나요? | 아니오 |

### Google Drive API 관련
앱이 Google Drive API를 사용하므로 다음을 추가 공지합니다:
- **목적**: 수업 사진·영상을 사용자 개인 Google Drive에 저장 및 조회
- **접근 범위**: `drive.file` (앱이 생성한 파일에만 접근, 전체 드라이브 접근 없음)
- **제3자 공유**: 없음 (사용자 본인 드라이브에만 저장)

## 4. 수출 규정 준수 (Export Compliance)

### Google Play
- **암호화 사용**: 예 (표준 HTTPS/TLS)
- **미국 수출 규정 적용**: 표준 암호화만 사용하므로 별도 ERN 불필요

### Apple App Store
App Store Connect > 버전 정보 > 수출 규정 준수에서 아래와 같이 답변합니다:

| 질문 | 답변 |
|------|------|
| 앱에 암호화 기술이 포함되어 있나요? | 예 |
| HTTPS/TLS 표준 암호화만 사용하나요? | 예 |
| 독점적이거나 비표준 암호화를 사용하나요? | 아니오 |
| 프랑스 앱 배포 시 CNIL 신고가 필요한가요? | 해당 없음 (한국 대상) |

**답변 결과**: 수출 허가 면제 대상 (ERN 불필요). `Info.plist`에 다음 키 추가:
```xml
<key>ITSAppUsesNonExemptEncryption</key>
<false/>
```

## 5. 아동·가족 프로그램 선언 (Made for Kids / Families)

- **아동 대상 앱인가요?** → 아니오

## 7. 주요 거절 사유 및 예방 방법

### Apple App Store 거절 사유

| 거절 사유 | 관련 가이드라인 | 예방 방법 |
|-----------|----------------|-----------|
| 개인정보처리방침 URL 누락 또는 미작동 | 5.1.1 | 심사 전 URL 접근 가능 여부 확인. 호스팅된 페이지 필요 (GitHub Pages 권장) |
| UGC 앱의 신고/차단 기능 미비 | 1.2 | 커뮤니티 게시글 신고 → 관리자 알림 → 숨김/삭제 플로우 구현 완료 확인 |
| 최소 기능 미달 (웹뷰 앱 의심) | 4.2 | Flutter 네이티브 앱임을 심사 노트에 명시. 주요 기능 동작 확인 |
| 앱 크래시 또는 버그 | 2.1 | 심사 제출 전 TestFlight 베타 테스트 필수 |
| 개인정보 라벨과 실제 수집 불일치 | 5.1.2 | 앱이 실제로 수집하는 데이터와 라벨이 정확히 일치하는지 검토 |
| 수출 규정 키 누락 | 수출 규정 | `Info.plist`에 `ITSAppUsesNonExemptEncryption = false` 설정 |
| 지원 URL 미작동 | 3.1.1 | 심사 전 지원 이메일 또는 페이지 URL이 정상 응답하는지 확인 |

### Google Play 거절 사유

| 거절 사유 | 정책 | 예방 방법 |
|-----------|------|-----------|
| 데이터 안전 섹션과 실제 수집 불일치 | 데이터 안전 정책 | 수집 데이터 항목을 앱 코드와 대조하여 정확히 기재 |
| 개인정보처리방침 미제공 또는 링크 오류 | 개인정보 보호 정책 | 공개 URL에 한국어 개인정보처리방침 게시 |
| 타겟 연령대 잘못 설정 | 가족 정책 | "아동용 앱" 체크 해제, 성인 대상 명시 |
| 민감한 권한 미사용 정당화 | 권한 정책 | 카메라/저장소 권한 사용 목적을 앱 설명 및 심사 노트에 기재 |

### 공통 예방 체크리스트
- [ ] 개인정보처리방침 URL이 공개적으로 접근 가능한지 확인
- [ ] 이용약관 URL이 공개적으로 접근 가능한지 확인
- [ ] 지원 이메일(contact@lionandthelab.com) 응답 가능 상태 확인
- [ ] 테스트 계정 3개(관리자/학부모/교사) 생성 및 동작 확인
- [ ] 앱 설명에 아동 직접 사용을 암시하는 표현 없는지 검토
- [ ] 모든 스크린샷이 현재 앱 버전과 일치하는지 확인
- [ ] 앱 내 '삭제 계정' 또는 '탈퇴' 기능이 정상 동작하는지 확인 (Apple 필수 요건)
