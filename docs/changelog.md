## [2.0.0] - 2026-03-14 (현재)

### 상용화 릴리즈

#### 신규 기능
- **네이티브 스플래시 스크린** — flutter_native_splash 적용
- **한국어 로케일** — Material/Cupertino 전역 한국어 적용
- **iOS 네이티브 전환** — CupertinoPageTransitionsBuilder 스와이프 백 제스처
- **촉각 피드백** — 탭 전환·버튼에 HapticFeedback 적용
- **설정/정보 페이지** — 계정 정보, 버전, 이용약관, 개인정보처리방침, 문의하기
- **앱 버전 표시** — 로그인 화면 + 설정 화면
- **스켈레톤 로딩** — 대시보드·부모뷰 로딩 상태 개선
- **Pull-to-refresh** — 5개 주요 탭 NestRefreshable 적용
- **빈 상태 안내** — 전체 탭 NestEmptyState 통일 적용
- **Android 뒤로가기** — PopScope + 더블 탭 종료

#### 버그 수정
- **로그인 빨간 화면** — ErrorWidget.builder를 모든 모드에서 적용, auth listener 중복 호출 방지
- **시간표 모바일 오버플로우** — 헤더 버튼 Wrap 레이아웃 적용, 그리드 가로 스크롤 padding 수정

#### 인프라
- **크래시 핸들링** — runZonedGuarded + 브랜드 에러 위젯
- **ListView 가상화** — 주요 3개 탭 ListView.builder 전환
- **반응형 레이아웃** — 1080px 기준 데스크톱/모바일 분기
- **NestTheme 전역 적용** — Warm Nest 컬러 팔레트 통일
- **Android AAB 빌드** — 47.8MB 릴리즈 번들
- **Bundle ID** — com.lionandthelab.nest
